# 8-puzzle

Solving 8-Puzzle using A\* Algorithm.

## Important to know:

- `0` is the empty tile

## Usage

### Running

Run in terminal:

> python3 puzzle.py

### Editing goal

Open the `goal.txt` file and edit the puzzle (3x3). Numbers need to be between 0 and 8.
